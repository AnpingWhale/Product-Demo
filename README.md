# Product-Demo

`Product-Demo` is the published demo repository for mature product prototypes.

## Structure

- `index.html`: demo hub landing page
- `coros-mcp/`: published COROS MCP demo
- `demos/`: local working copies for mature demos before publishing
- `labs/`: experiments and half-finished concepts
- `shared/`: shared assets, scripts, and templates

## Current demo

- `coros-mcp/`
